How to install this signature:
  1. Press Cmd+A (Mac) or Ctrl+A (Windows) to select everything on this page
  2. Press Cmd+C / Ctrl+C to copy
  3. Gmail: Settings ⚙️ → See all settings → General → Signature → paste with Cmd+V / Ctrl+V → Save Changes
  4. Outlook Desktop: File → Options → Mail → Signatures → New → paste → OK
  5. Outlook 365 web: Settings → Mail → Compose and reply → paste → Save
  6. Apple Mail: Mail → Settings → Signatures → + → paste into the right panel
