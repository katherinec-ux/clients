How to set up your email signature

  1. Click the Copy Signature button next to your name below — it copies the formatted signature to your clipboard
  2. If the button says "Selected — press Cmd/Ctrl+C", your browser blocked auto-copy — just press Cmd+C (Mac) or Ctrl+C (Windows) while the signature is highlighted
  3. Gmail: Settings (⚙️ gear) → See all settings → General → Signature → create/edit your signature → paste with Ctrl+V / Cmd+V → Save Changes at the bottom
  4. Outlook Desktop: File → Options → Mail → Signatures → New or select existing → paste into the editor box → OK
  5. Outlook 365 (web): Settings → Mail → Compose and reply → Email signature → paste → Save
  6. Apple Mail: Mail → Settings → Signatures → select account → + → paste into the right-hand field

Tip: Make sure to paste using the keyboard shortcut (Ctrl+V / Cmd+V), not right-click → Paste, as right-click paste may strip formatting in some email clients.
